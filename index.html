Currently, I mainly contribute to open source projects, who need a rockstar developer to fix important bug reports. My dad showed me how to turn on the computer once and since that moment I started to speak code. I never stopped. SKILLS

Javascript

JavaScript is one of the three core technologies of World Wide Web content production. It is used to make webpages interactive and provide online programs, including video games.

JQuery

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It is free, open-source software using the permissive MIT License. Web analysis indicates that it is the most widely deployed JavaScript library by a large margin.

React

React allows developers to create large web-applications that use data and can change over time without reloading the page. It aims primarily to provide speed, simplicity, and scalability. React processes only user interfaces in applications.

Angular

Angular is a framework for building client applications in HTML and either JavaScript or a language like TypeScript that compiles to JavaScript. Angular empowers developers to build applications that live on the web, mobile, or the desktop.
